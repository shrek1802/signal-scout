# 📡 Signal Scout

Signal Scout is an Android application designed to help installers and home users achieve the best possible 4G/5G signal from Huawei LTE/5G routers.

Unlike traditional router apps, Signal Scout guides the user through the installation process, helping them align antennas, optimise band selection, and improve overall connection quality.

---

## 🚀 Features

### Current
- Router login
- Live SINR, RSRP, RSRQ and RSSI monitoring
- Live signal quality display
- Audio alignment beeps
- Best signal tracking
- Huawei router support

### Planned
- 📡 Automatic band scanner
- 🔒 Band locking
- 📶 Cell locking
- 🧭 Tower finder (AR camera mode)
- 📈 Live signal graphs
- 📄 Installer report (PDF)
- ⭐ Signal Quality Score (0–100)
- 🚀 Automatic optimisation wizard
- 🌍 Multi-router support

# 🛠 Installation Wizard

Signal Scout guides you through every stage of an installation.

1. Connect to the router
2. Aim the antenna
3. Scan all supported bands
4. Find the best carrier aggregation
5. Lock the best combination
6. Generate an installation report

# ❤️ Why this project exists

Most router utilities expose technical information but provide little guidance.

Signal Scout focuses on helping users achieve the best possible installation by combining:

- Live diagnostics
- Guided setup
- Automatic optimisation
- Installer-friendly tools
